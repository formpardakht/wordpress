اسکریپت فرم پرداخت پِی

== Description ==

اسکریپت فرم پرداخت یک اسکریپت ساده و پر کاربرد در زمینه پرداخت آنلاین می باشد که با استفاده از آن می توانید فرم های پرداخت متنوع با فیلد های سفارشی ایجاد کنید و از لینک فرم ساخته شده را با مشتری هایتان به اشتراک بگذارید. همچنین میتوانید تراکنش ها را مدیریت کنید و حتی آمار زنده درآمد و تراکنش ها را از طریق بخش مدیریت ساده و زیبای اسکریپت فرم پرداخت مشاهده کنید. این اسکریپت دارای امکانات متنوع دیگری نیز می باشد که در ادامه به برخی دیگر از آن ها به صورت کلی نیز اشاره می کنیم

= مشخصات =
* امکان ایجاد تعداد بی نهایت فرم پرداخت
* اضافه و یا حذف کردن فیلد های فرم
* اضافه و یا حذف کردن فیلد های فرم
* بارگذاری تصویر برای هر فرم
* ایجاد محدودیت تعداد دفعات پرداخت برای فرم
* آمار زنده تراکنش ها
* آمار زنده درآمد
* ایجاد فاکتور پرداخت
* ارسال لینک فاکتور در شبکه های اجتماعی
* نمودار درآمد
* لیست تراکنش ها
* سفارشی سازی پوسته
* شبکه پرداخت پِی
* بروز رسانی به نسخه های جدید
* بسته نصب آسان
* سیستم فروش فایل (جدید)
* اضافه کردن اسکریپت و استایل سفارشی به سایت (جدید)

* PHP version 5.2.4 or greater
